# NPM

## NPM useful commands

| command                    | description                                     |
|----------------------------|-------------------------------------------------|
| `npm -v`                   | Show NPM version                                |
| `npm install`              | Install all                                     |
| `npm install <package>`    | Install package locally                         |
| `npm install -g <package>` | Install package globally                        |
| `npm uninstall`            |                                                 |
| `npm remove`               |                                                 |
| `npm update`               |                                                 |
| `npm start`                |                                                 |
| `npm audit`                |                                                 |
| `npm outdated`             |                                                 |
